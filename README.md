# news-recommend
# 项目架构
系统采用Django框架作为后端开发基础搭配，结合BeautifulSoup库进行新闻数据的爬取与预处理。前端界面利用Bootstrap框架+sweetalert进行构建，结合jQuery和AJAX进行页面异步通信的动态网页设计，引入echarts可视化数据图表进行数据可视化处理。同时借助MySQL数据库管理系统存储用户数据、新闻数据以及用户兴趣偏好等信息。
# 推荐算法
推荐引擎是基于权重融合的推荐算法
